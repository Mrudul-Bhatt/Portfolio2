MERN Chat App

App rooms for chatting

This application contains rooms for chatting for users. User of a particular room is isolated from other room user

Front end of app is made with ReactJs framework and deployed on Netlify, it serves two pages. One page is for signing up user with display name and room name, which then redirects user to chat interface. Users of different rooms are isolated from each other

Backend of app is written in NodeJs which has helper functions, currently the project doesn't use MongoDb but will be added soon, it is deployed on Heroku
