NodeJS E-Commerce App

App to buy and sell products

E-Commerce website for buying and selling products with Login & Signup. Users need to register in order to use complete functionality of site

User Interface of application is served using templating engine EJS, no frontend framework is used except view engines.

App is supported by a NodeJS backend which performs all the CRUD operations MongoDB is used as a database for storage, there are cart and order modules along with register and login, add, update and delete product cards, payment functionality to be added soon. Routes are protected and cannot be reached without registering. App is deployed on Heroku
