MERN QuizApp

A Quiz app

An app which loads random questions from API and displays them with 4 options of which 1 is correct

Frontend is made with ReactJs which uses endpoints url for fetching from API, user chooses answers and score is seen after attempting all questions with play again option. Frontend is deployed on Netlify

NodeJs is used for creating API which has questions and array of answers, databse used is MongoDB. In order to alter questions or answers you need postman or insomnia pseudo interfaces to communicate with API. API is deployed on Heroku
