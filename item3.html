MERN Schedule-Tracker

App for managing of exercises

An app for adding users and managing their schedule of exercises and work

Interface is made using ReactJs deployed on Netlify. Communication with backend api is met with axios

Backend is made using NodeJs with CRUD operations implementation, for creating user, adding schedule, updating it, deleting it, setting time and date for schedule. MongoDB is used for database, app deployed on Heroku
